## Build
1. Clone this repository.
```
git clone https://github.com/vikdevelop/duckduckgo-browser.git
```
2. Build this program with CMake & make.
```
cd duckduckgo-browser &&
cmake -B build &&
cd build && make
```
## Depencies
```
cmake
gtkmm-3.0
webkit2gtk
```
