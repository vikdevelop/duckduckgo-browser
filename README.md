# DuckDuckGo Browser
WebBrowser with DuckDuckGo &amp; WebKitGTK
## Build
1. Clone this repository.
```
git clone https://github.com/vikdevelop/duckduckgo-browser.git
```
2. Build this program with Cmake & make.
```
cd [repository name] &&
cmake -B build &&
cd build && make
```
## Depencies
```
cmake
gtkmm-3.0
webkit2gtk
```
